循环语句，故名思意就是让代码形成一个回路，重复执行。
JavaScript中的循环语句只有for语句和while语句
for(初始值；布尔；计数器){
	//语句块
}
for语句中如果布尔值是turn，就会一直执行，如是fase则不循环，所以为防止死循环
添加一个计数器，让布尔值变成false，循环停止。
for(let i = 0 ; i<10;i++){
	//i的初始值是0
	//判断i是否小于10，如果小于10则执行花括号里的代码
	//每次执行花括号中的代码，i值加1
	console.log(i)；
}
代码只会输出0-9的数字；到10则i不<10所以不执行；
还可以通过for循环来计算0-10的加和：
var sum = 0;                 //sum用来存储循环过程中正整数的加和
for(var i = 1;i<=10;i++){
    sum += i;
}
console.log(sum);
while语法不常用和for循环类似；
while(bool){
	//bool为trun，循环执行
}；省去了初始值和计数器但为了防止死循环，还是的后写；
continue：
可以结束本次循环，直接进入下一个循环；例如：
while(i<10){
if(i=2){
	continue
};//这样在i等于2时候跳过
}；
break:
结束语句当循环到指定break时候结束循环。
